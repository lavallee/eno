# Active Recall

Active recall is the practice of retrieving information from memory rather than
re-reading it. Closing the book and reconstructing an answer is uncomfortable, and
that discomfort is precisely the signal that learning is happening.

Pair it with [[Spaced Repetition]] to decide when to test, and with [[Zettelkasten]]
notes phrased as questions to make testing frictionless. Reading in the
[[Marginalia]] of a book is a lightweight on-ramp: a margin question today is a recall
prompt tomorrow. Sustained over weeks it becomes a form of [[Deliberate Practice]].
